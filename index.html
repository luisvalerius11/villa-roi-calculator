Data sourcing & limitations

Villa comparables. This in-browser tool ships with editable mock comps. To add a comp from the web, paste a listing URL — the app creates a blank row prefilled with the URL so you can manually transfer ask price, beds, ADR and notes. Direct fetching of listing sites is typically blocked by CORS in pure browser code; a thin server-side proxy or scraping API (Apify, Bright Data, Scrapy Cloud, or a custom Cloudflare Worker) can be added to populate fields automatically.

Yield benchmark data

Benchmarks ship as editable defaults per market (Singapore, Australia, UK, France, Netherlands, Spain, Hong Kong, UAE, Indonesia). All values can be overridden inline. To wire live rates, attach: SGS / SG T-bills (MAS API), Australian Government Bonds (AOFM / RBA), UK gilts (DMO), Eurozone sovereigns (ECB SDW), HKMA EFBN, UAE/Dubai bonds (DFM API). REIT dividend yields can be sourced via FTSE EPRA NAREIT, iEdge S-REIT, or vendor APIs (Bloomberg, Refinitiv, EOD Historical). Bank deposit rates change too frequently for stable APIs — schedule a daily refresh job or maintain a manual board.

Comparison philosophy

This tool only benchmarks against income-oriented yield comparators that an income investor would actually consider: deposits, government bonds, T-bills, money market funds, REIT dividend yields, sukuk, and local property income funds. It deliberately omits equity indices, growth stocks, and venture-style return benchmarks because those are not apples-to-apples with property cash-yield investing. The relevant question this dashboard answers is: does the villa earn a sufficient premium over conservative income alternatives to justify the illiquidity, operational burden, vacancy risk and CapEx burden of owning the asset?

Browser-only limits

This app is fully client-side. There is no backend, no cookies, no telemetry. All numbers are computed locally with JavaScript. To extend with live data, the easiest path is a thin server proxy (Cloudflare Worker / Vercel Edge Function) that exposes a few normalized endpoints — comp listings, sovereign yields, deposit rates, REIT dividend yields — which this UI can call directly.
